DATABASE PPJB-KOCE
desain database personal user untuk mengelolah data panen, pengeringan, dan jual-beli milik pedagang atau petani
DATABASE CLOVE COMODITY
desain database multi user untuk mengelolah data panen, pengeringan, dan jual-beli milik pedagang/petani dan sistem informasi terintegrasi bersama buruh petik cengkeh
API CLOVE COMODITY
adalah sistem pengelolan data komoditas cengkeh dari proses panen , proses pengeringan, dan proses jual-beli. Pada prinsipnya fungsi dari sistem pengelolah data ini di peruntukkan untuk elemen yang terlibat usaha komoditas diantaranya: petani, pedagang atau pengusaha, serta buruh. Pengguna API diklasifikasikan menjadi 3 jenis, antara lain: umum, front-end untuk user utama ACC(Aplication Clove Comodity), dan front-end untuk user mitra/pegawai/pekerja. Semua metode dan titik akhir yang memerlukan otorisasi ditujukan untuk aplikasi front-end komoditas cengkeh (acc) dan mitra/karyawan/'pekerja' front-end.
