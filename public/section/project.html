API CLOVE COMMODITY

Sistem pengelolaan data komoditas cengkeh dari proses panen, pengeringan, dan jual-beli. Pada prinsipnya, fungsi dari sistem pengelolaan data ini diperuntukkan bagi elemen yang terlibat dalam usaha komoditas, di antaranya: petani, pedagang atau pengusaha, serta buruh. Pengguna API diklasifikasikan menjadi 3 jenis, antara lain: umum, front-end untuk user utama ACC (Application Clove Commodity), dan front-end untuk user mitra/pegawai/pekerja. Semua metode dan titik akhir yang memerlukan otorisasi ditujukan untuk aplikasi front-end komoditas cengkeh (ACC) dan mitra/karyawan/'pekerja' front-end. Untuk detail lebih lanjut, lihat nama grup. Untuk endpoint lokasi, dapat digunakan secara bebas untuk umum.
